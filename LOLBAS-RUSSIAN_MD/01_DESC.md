| AWL Bypass                                                                                                                                                                                                                                                                                                                                                                        | Обход белого списка приложений (Application Whitelist Bypass). Использование доверенных, подписанных бинарных файлов (например, msbuild.exe, regsvr32.exe) для выполнения вредоносного кода в обход политик AppLocker, WDAC или аналогичных систем, разрешающих запуск только утверждённых программ.                                                                                                                                                                                      |
| Copy                                                                                                                                                                                                                                                                                                                                                                              | Копирование. Использование системных утилит для копирования или перемещения файлов (включая вредоносные), часто в рамках подготовки к атаке, перемещения по сети или эксфильтрации данных (например, xcopy, robocopy).                                                                                                                                                                                                                                                                    |
| Credentials                                                                                                                                                                                                                                                                                                                                                                       | Работа с учётными данными. Использование инструментов для кражи, дампа, извлечения или манипуляции паролями, хешами, билетами Kerberos и другими токенами аутентификации (например, с помощью rundll32.exe для вызова функций DPAPI, reg.exe для экспорта хешей SAM).                                                                                                                                                                                                                     |
| Decode                                                                                                                                                                                                                                                                                                                                                                            | Декодирование. Преобразование закодированных или зашифрованных данных в читаемый или исполняемый формат. Например, использование certutil.exe для декодирования Base64-файла обратно в бинарный.                                                                                                                                                                                                                                                                                          |
| Download                                                                                                                                                                                                                                                                                                                                                                          | Загрузка файлов. Использование легитимных сетевых клиентов или утилит для скачивания вредоносных payload'ов с удалённых серверов на целевую машину (например, bitsadmin.exe, certutil.exe, powershell.exe).                                                                                                                                                                                                                                                                               |
| Dump                                                                                                                                                                                                                                                                                                                                                                              | Дамп памяти или данных. Использование системных или отладочных инструментов для извлечения конфиденциальной информации из памяти процессов (например, LSASS для получения учётных данных), файлов или реестра.                                                                                                                                                                                                                                                                            |
| Encode                                                                                                                                                                                                                                                                                                                                                                            | Кодирование. Преобразование данных или бинарных файлов в закодированный формат (например, Base64) для сокрытия содержимого при передаче через средства мониторинга или для подготовки к выполнению в памяти.                                                                                                                                                                                                                                                                              |
| Execute                                                                                                                                                                                                                                                                                                                                                                           | Выполнение кода. Основная категория, означающая запуск произвольного кода, скриптов или команд. Часто связана с техниками "Living off the Land", когда для выполнения используется доверенный бинарник как прокси (например, запуск скрипта через wscript.exe или DLL через rundll32.exe).                                                                                                                                                                                                |
| Reconnaissance                                                                                                                                                                                                                                                                                                                                                                    | Разведка. Сбор информации о системе, сети, пользователях, установленном ПО, конфигурациях безопасности и активных процессах перед эскалацией привилегий или перемещением по сети (например, с помощью whoami.exe, systeminfo.exe, nltest.exe, net.exe).                                                                                                                                                                                                                                   |
| Tamper                                                                                                                                                                                                                                                                                                                                                                            | Изменение/Манипуляция. Намеренное изменение конфигураций системы, реестра, служб, политик безопасности или журналов событий для отключения защитных механизмов, обеспечения персистентности или сокрытия следов.                                                                                                                                                                                                                                                                          |
| UAC Bypass                                                                                                                                                                                                                                                                                                                                                                        | Обход контроля учётных записей (UAC). Использование уязвимостей или особенностей в доверенных компонентах Windows для повышения привилегий от обычного пользователя до администратора без запроса пароля через UAC-окно (например, с помощью fodhelper.exe, sdclt.exe, eventvwr.exe).                                                                                                                                                                                                     |
| Upload                                                                                                                                                                                                                                                                                                                                                                            | Выгрузка файлов. Использование системных инструментов для отправки (экспорта) данных с компрометированной машины на контролируемый злоумышленником сервер в рамках эксфильтрации информации (например, через curl.exe, ftp.exe, powershell).                                                                                                                                                                                                                                              |
| ADS                                                                                                                                                                                                                                                                                                                                                                               | Работа с альтернативными потоками данных NTFS (Alternate Data Streams). Использование возможностей файловой системы NTFS для скрытия вредоносных файлов, скриптов или данных внутри альтернативных потоков других файлов или папок (например, file.txt:evil.exe). Используется для сокрытия артефактов от пользователя и средств защиты, а также для обхода политик, ограничивающих выполнение файлов по расширению. Например, выполнение скрипта, скрытого в ADS, с помощью wscript.exe. |
| Compile                                                                                                                                                                                                                                                                                                                                                                           | Компиляция кода. Использование системных компиляторов или инструментов разработки (например, csc.exe, msbuild.exe) для преобразования исходного кода (C#, C++ и т.д.) во вредоносный исполняемый файл непосредственно на целевой системе, что позволяет обойти защиту, не загружая готовые бинарники.                                                                                                                                                                                     |
| Conceal                                                                                                                                                                                                                                                                                                                                                                           | Сокрытие. Использование возможностей ОС для маскировки вредоносной активности или данных (например, скрытие файлов в альтернативных потоках данных NTFS (ADS), использование атрибутов "скрытый" или "системный", работа с реестром).                                                                                                                                                                                                                                                     |
